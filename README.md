# Padelero — Frontend

Aplicación frontend para buscar partidos de pádel y concertar encuentros. Permite ver horarios, apuntarse a partidos y enviar un mensaje al organizador para cerrar el acuerdo.

Acceso a la aplicación: https://padelero.es

Usuario de prueba

- Usuario: `test`
- Contraseña: `12345qwerT`

Tecnologías principales

- Vite
- React 18
- React Router (react-router-dom)
- Redux Toolkit
- Material UI (@mui)
- react-big-calendar
- Axios para llamadas HTTP
- moment / moment-timezone
- Notistack / react-toastify para notificaciones

## Descripción breve del flujo

- Buscar partidos disponibles en el calendario.
- Enviar solicitud o mensaje al organizador para cerrar los detalles del partido (hora, pista, confirmación).
- La app facilita la comunicación rápida entre jugadores para coordinar y confirmar encuentros.
